# src (Server)

Source code for the Node/Express backend. Organised by feature modules:

* `config/` – Configuration helpers such as DB connection
* `routes/` – Express routers
* `controllers/` – Request handlers
* `services/` – Business logic
* `models/` – Mongoose schemas
* `middleware/` – Custom Express middlewares
* `chat/` – Prompt templates & chat utilities
* `utils/` – General-purpose helpers & seed scripts
* `index.js` – Application entry that wires everything together
